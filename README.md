## 💻 Projeto
Esse é um projeto Web Responsivo de uma calculadora IMC para calcular a gordura corporal.
## 🚀 Tecnologias
Esse projeto foi desenvolvido durante um vídeo no Youtube com as seguintes tecnologias:

- HTML
- CSS
- JavaScript
- Git e Github
